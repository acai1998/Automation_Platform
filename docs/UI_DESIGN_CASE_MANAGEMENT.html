5. 设计要点说明

✓ 页面拆分，能力复用

三个页面使用相同的 BaseCaseList 组件，通过配置实现不同展示效果

✓ 统一的交互模式

所有页面都包含：同步用例按钮、搜索功能（性能页面除外）、列表展示、分页控制

✓ 视觉区分

不同页面使用不同的主题色（蓝色-API、紫色-UI、橙色-性能）便于识别

✓ 状态展示

用例状态通过徽章展示（idle/running），运行中时按钮禁用

✓ 响应式设计

支持桌面端和移动端，搜索框和按钮自适应布局
