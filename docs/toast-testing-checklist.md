## 🐛 常见问题排查

### 问题 1：Toast 不显示

**可能原因**：
- `Toaster` 组件未在 `App.tsx` 中引入
- 导入路径错误
- 组件渲染错误

**排查步骤**：
1. 检查浏览器控制台是否有错误
2. 确认 `src/App.tsx` 中包含 `<Toaster />`
3. 确认 `execution-toast.tsx` 文件存在
4. 运行 `npx tsc --noEmit` 检查类型错误

---

### 问题 2：点击"查看记录"没有跳转

**可能原因**：
- `runId` 为 undefined 或 null
- 路由配置问题
- `result` 对象结构不正确

**排查步骤**：
1. 在浏览器控制台查看 `result` 对象：
   ```javascript
   console.log('Execution result:', result);
   ```
2. 确认 `result.runId` 有值
3. 检查路由 `/reports/:id` 是否已配置
4. 手动访问 `/reports/123` 测试路由

---

### 问题 3：样式显示异常

**可能原因**：
- Tailwind CSS 未正确编译
- 暗黑模式类名冲突
- 全局样式覆盖

**排查步骤**：
1. 重启开发服务器：`npm run dev`
2. 清除浏览器缓存
3. 检查 `tailwind.config.js` 配置
4. 在浏览器开发者工具中检查元素样式

---

### 问题 4："查看 Jenkins"按钮不显示

**可能原因**：
- `buildUrl` 为空或 undefined
- Jenkins 未配置

**排查步骤**：
1. 检查 `result.buildUrl` 的值
2. 确认 Jenkins 配置正确
3. 查看后端日志确认 Jenkins 触发成功

---

### 问题 5：重试功能不工作

**可能原因**：
- `onRetry` 回调未正确传递
- 函数闭包问题

**排查步骤**：
1. 在浏览器控制台查看是否有错误
2. 确认 `handleRunCase` 函数参数正确
3. 添加 `console.log` 调试：
   ```typescript
   onRetry: () => {
     console.log('Retry clicked:', caseId, caseName, projectId);
     handleRunCase(caseId, caseName, projectId);
   }
   ```
